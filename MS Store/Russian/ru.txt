Описание:
Бесплатное приложение с открытым исходным кодом для установки GIF, видео и веб-страниц в качестве обоев рабочего стола.
Благодаря браузеру Chromium и MPV-плееру ваши обои поддерживают новейшие веб-технологии и стандарты видео.

Это проект с открытым исходным кодом, и участие сообщества приветствуется. Для участия, пожалуйста, посетите
https://github.com/rocksdanister/lively

Функции:
Воспроизведение обоев приостанавливается (использование ресурсов 0%) при запуске игр/работе.
Поддерживает несколько дисплеев с любым соотношением сторон и разрешением.
Аппаратное ускорение воспроизведения видео с помощью mpv-плеера, установка любого видео в качестве обоев.
С помощью Chromium Embedded Framework (CEF) установите любой веб-сайт в качестве обоев.
Используйте Lively Audio API, чтобы создавать обои, которые реагируют на исходящие звуки.
Используйте Lively System API для создания виджетов, отображающих системную информацию.
Поддерживает онлайн-видеопотоки и ссылки Shadertoy.com.
Полностью открытый и бесплатный, без платного доступа.
Поддерживает темную и светлую тему.
