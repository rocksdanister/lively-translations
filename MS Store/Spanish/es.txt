Descripción:
Aplicación gratis y de código abierto para aplicar GIFs, videos y páginas web como tu fondo de pantalla.
Usando el motor del navegador web Chromium y el reproductor MPV, tus fondos de pantalla tendrán el soporte de los últimos estándares de video y tecnologías web.

Este proyecto es de código abierto y la participación de la comunidad es siempre bienvenida. Para participar por favor visita
https://github.com/rocksdanister/lively

Características:
La animación del fondo de pantalla es pausada (0% recursos usados) cuando te encuentras jugando/trabajando.
Soporta múltiples pantallas de cualquier tamaño y resolución.
Asigna cualquier video como tu fondo de pantalla.
Asigna cualquier página web como tu fondo de pantalla (Motorizado por Chromium Embedded Framework).
Usa Lively Audio API para hacer que tus fondos reaccionen al audio del sistema.
Usa Lively System API para crear widgets que muestren información del sistema.
Soporte para la reproducción de video acelerado por hardware (Motorizado por el reproductor MPV).
Soporte para reproducción de videos en vivo y enlaces a Shadertoy.com.
Completamente abierto y gratis. Ninguna característica es pagada.
Soporta tanto el modo claro como el modo oscuro.
